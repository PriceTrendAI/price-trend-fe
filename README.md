## Getting Started

**Check versions**

```
$ node --version
v22.14.0

$ npm --version
11.3.0
```

**Install dependencies**

```
$ npm install
```

**Setup Environment Variables**

```
$ cp .sample.env .env
```

**Run Development Server**

```
$ npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
